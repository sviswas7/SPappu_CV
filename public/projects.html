Tic Tac Toe

  • Developed a simple game using Java and C++, implemented methods and logic to print a Game Board, allowing the user to interact with the program, with set winning, losing, and tie conditions.

Dictionary

  • This program allows to create a dictionary, where the user can add words and their definitions which are pushed into a vector. A user interactive menu is implemented allowing to view the word definitions, along with full list of words.

Bank Account

  • This project is developed in C++ using object-oriented programming, which allows users to check the balance, withdraw, deposit currency into their bank accounts. A Bank Account class is created with 2 private identifiers and 6 member functions.

RPG Player

  • The project is created using foundational classes to represent the Player Character, that is, the entity within the game world that the user would control. There are three derived classes, Warrior, Priest, Mage accordingly. The main function allows the user to create different kinds of Player objects, allowing them to select the derived class, which request as a profession from the user from a simple menu.
